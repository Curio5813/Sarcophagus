Shadow Warrior - The Last Samurai

Shadow Warrior, a first-person shooter originally released in 1997 for PC, has been ported to other systems in several ways. Recently, a free version of the game, called Shadow Warrior Classic, was released on Steam. This version contains the same gameplay as before, with slight changes to the controls but no changes to the visuals. Check out our review of this classic game.
